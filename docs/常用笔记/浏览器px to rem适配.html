дҪҝз”Ёpostcss-px2remйҖӮй…Қе·Із»ҸеҒҡжҲҗеҠҹ1920*1080йҖӮй…Қдёәrem

иҰҒдҪҝpostcss-px2remз”ҹж•ҲпјҢжӮЁйңҖиҰҒзЎ®дҝқе·Із»ҸеңЁйЎ№зӣ®дёӯе®үиЈ…дәҶpostcss-loaderе’Ңpostcss-px2remгҖӮз„¶еҗҺпјҢжӮЁйңҖиҰҒеңЁwebpack.config.jsдёӯй…ҚзҪ®е®ғд»¬гҖӮд»ҘдёӢжҳҜдҝ®ж”№еҗҺзҡ„зӨәдҫӢпјҡ

  1. йҰ–е…ҲпјҢзЎ®дҝқе·Із»Ҹе®үиЈ…дәҶpostcss-loaderпјҡ

InsertCopy

npm install postcss-loader --save-dev
  1. з„¶еҗҺпјҢеңЁwebpack.config.jsдёӯй…ҚзҪ®postcss-loaderпјҡ

InsertCopy

const path = require('path');

module.exports = {
 // ...
 module: {
   rules: [
     // ...
     {
       test: /\.css$/,
       use: [
         'vue-style-loader',
         'css-loader',
         'postcss-loader' // ж·»еҠ жӯӨиЎҢ
       ]
     },
     // ...
   ]
 },
 // ...
};
  1. жҺҘдёӢжқҘпјҢзЎ®дҝқе·Із»Ҹе®үиЈ…дәҶpostcss-px2remпјҡ

InsertCopy

npm install postcss-px2rem --save-dev
  1. еңЁpostcss.config.jsдёӯй…ҚзҪ®postcss-px2remпјҡ

InsertCopy

module.exports = {
 plugins: {
   autoprefixer: {},
   'postcss-px2rem': {
     remUnit: 16, // и®ҫзҪ®remеҚ•дҪҚдёә16
     viewBox: false, // е…ій—ӯviewBox
     selectorBlackList: ['.ignore', '.hairlines'] // жҺ’йҷӨзү№е®ҡйҖүжӢ©еҷЁ
   }
 }
};
  1. зЎ®дҝқеңЁvue.config.jsдёӯй…ҚзҪ®дәҶcss.loaderOptionsпјҡ

InsertCopy

   loaderOptions: {
     postcss: {
       plugins: [
         // ...
         require('postcss-px2rem')({
           remUnit: 16,
           viewBox: false,
           selectorBlackList: ['.ignore', '.hairlines']
         })
         // ...
       ]
     }
   }
 }
};

зҺ°еңЁпјҢpostcss-px2remеә”иҜҘе·Із»Ҹз”ҹж•ҲдәҶгҖӮ

еҰӮжһңжӮЁе·Із»ҸжҢүз…§дёҠиҝ°жӯҘйӘӨй…ҚзҪ®дәҶpostcss-px2remпјҢдҪҶд»Қз„¶жІЎжңүзңӢеҲ°ж•ҲжһңпјҢйӮЈд№ҲеҸҜиғҪжҳҜеӣ дёәд»ҘдёӢеҺҹеӣ пјҡ

  1. зЎ®дҝқжӮЁзҡ„postcss.config.jsж–Үд»¶дҪҚдәҺйЎ№зӣ®зҡ„ж №зӣ®еҪ•дёӢгҖӮ
  2. зЎ®дҝқжӮЁзҡ„webpack.config.jsж–Үд»¶д№ҹдҪҚдәҺйЎ№зӣ®зҡ„ж №зӣ®еҪ•дёӢгҖӮ
  3. зЎ®дҝқжӮЁзҡ„vue.config.jsж–Үд»¶д№ҹдҪҚдәҺйЎ№зӣ®зҡ„ж №зӣ®еҪ•дёӢгҖӮ
  4. зЎ®дҝқжӮЁзҡ„package.jsonж–Үд»¶дёӯеҢ…еҗ«дәҶpostcss-loaderе’Ңpostcss-px2remзҡ„дҫқиө–гҖӮ
  5. жё…йҷӨжөҸи§ҲеҷЁзј“еӯҳпјҢд»Ҙдҫҝж–°зҡ„й…ҚзҪ®з”ҹж•ҲгҖӮ

еҰӮжһңй—®йўҳд»Қз„¶еӯҳеңЁпјҢиҜ·жЈҖжҹҘжҺ§еҲ¶еҸ°иҫ“еҮәдёӯжҳҜеҗҰжңүе…ідәҺpostcss-px2remзҡ„й”ҷиҜҜдҝЎжҒҜгҖӮ
